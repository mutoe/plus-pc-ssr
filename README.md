# ThinkSNS+ Desktop Client

> **(prototype)** VUE SSR version for ThinkSNS+ Desktop View.

## Build Setup

``` bash
# 安装依赖
$ yarn

# 启动开发环境
$ yarn dev

# 以生产环境构建并启动服务
$ yarn build
$ yarn start

# 构建 SPA 静态版本
$ yarn generate

# 启动测试
$ yarn test

```
